## 🛠️ System Requirements

CommandNest runs on modern Windows systems. Ensure your machine meets these basic requirements for the best experience:

* Windows 10 or Windows 11.
* An active internet connection to access AI models.
* At least 4GB of system memory.
* A screen resolution of 1280x720 or higher.
* An account with OpenRouter to manage your AI provider connections.

## ⚙️ How It Works

CommandNest functions as a bridge between your desktop environment and powerful AI models. 

### Global Hotkeys
Access the assistant from any program. Press the designated hotkey combination on your keyboard to bring the input window to the front. This allows you to ask questions or issue commands while you work in other applications like browsers or text editors.

### Streaming Results
The application displays answers as they generate. This mimics how a human types, which allows you to read information as it appears rather than waiting for the entire block of text to finish.

### Model Context Protocol
CommandNest uses the Model Context Protocol to understand your local files. This provides the AI with details about your current work, which leads to more accurate answers. You can grant access to folders or specific documents to allow the assistant to analyze your content.

## 🔑 Setting Up Your AI Connection

This application acts as a gateway. To use the AI features, you must provide your own connection details.

1. Open CommandNest.
2. Navigate to the Settings menu.
3. Find the field labeled API Key.
4. Log into your OpenRouter account to generate a new key.
5. Paste that key into the designated box within the app.
6. Save your settings to begin using the service.

Your API key acts as a digital credit card for the AI service. The AI provider charges small fees based on the amount of text processed.

## 📝 Troubleshooting

If the application fails to start, check these common items:

* **Internet Connection:** Ensure your firewall allows CommandNest to access the internet.
* **API Key:** Verify your OpenRouter key is active and has sufficient funds stored in your account.
* **Updates:** Check the release page regularly. Version updates fix bugs and improve performance.
* **Keyboard Conflicts:** If the global hotkey does not trigger the window, check if another application uses the same shortcut. You can change your hotkey settings inside the app menu.
